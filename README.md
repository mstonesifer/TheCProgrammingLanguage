# TheCProgrammingLanguage

## Purpose

A repo for completing exercises and projects presented in "The C Programming Language 2nd Edition" by Brian Kernighan and Dennis Ritchie.

## Organization

- `/ch[n]` folders labels reflect the chapter from which the code was pulled from / written for.
- `/Examples` are programs copied directly from the book (potentially with small modifications in order to get snippets running).
- `/Exercises` are my best effort attempt at solving each problem with the knowledge I had at the time of reading.
  - Nearly all examples are at least working based on my interpretation of the problem, with the understanding I cannot ask the writers clarifying questions.
  - Skipped, unfinished or incomplete solutions, few as they may be, are likely due to "life" happening, and time being a limited resource, and just needing to get on with it (I have 4 more 'C' books on the shelf I would like to get through c:).
  - Per-chapter solutions are not to be modified directly at a later time. Rather, if a problem is to be resolved, it should be added to a separate folder to memorialize the progress that is made during study. 
- `/sandbox` small tests or info programs
- `/templates` used for storing scaffold files
- `/testdata` quick and dirty test files to feed (mostly just pipe into) programs for testing

## Rationale (What is this really for?)

- We all like seeing progress, and this here is to document mine as I learn 'C' with no prior experience with the language.

## Disclaimer

- Code is potentially incomplete, likely unsafe, or otherwise not intended for production use in any capacity.
- Everything here is strictly to document my learning journey as I learn C using the aforementioned book.
- I make no claim that any solutions found here are correct; However, I do check each program does run at a minimum and meets the basic requirements outlined by each exercise completed.
